# Telecom Customer Churn Prediction

> **Type:** ML Model  
> **Author:** Prajwal Markal Puttaswamy  
> **Stack:** Python · XGBoost · SHAP · Scikit-learn · Pandas · Matplotlib

## Overview

End-to-end churn prediction pipeline using XGBoost on telecom data. Achieves 89% AUC with SHAP-based feature explainability.

## Tech Stack

- Python
- XGBoost
- SHAP
- Scikit-learn
- Pandas
- Matplotlib

## Getting Started

```bash
# Clone the repo
git clone https://github.com/prajwal467262/telecom-customer-churn-prediction

# Install dependencies
pip install -r requirements.txt

# (If applicable) Generate sample data first
python generate_sample_data.py

# Run the main script
python telecom_customer_churn.py
```
